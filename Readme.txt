Environment Dependency:
This code is written on java 8 version and writeen inside package having name "project" .

All input Files and output files will be created in Location - /D://


To Run the code:
First run the InsertionSort.java - this will create the 4 input files and reads it, sort it and create 4 output files which are sorted using insertion sort with time taken for this sorting.
Second run the MergeSort.java -  this will reads the four files, sort it and create 4 output files which are sorted using insertion sort with time taken for this sorting.
Last run the QuickSort.java - this will reads the four files, sort it and create 4 output files which are sorted using insertion sort with time taken for this sorting.
